## ğŸš€ Usage

1. Clone this repository:
   ```bash
   git clone https://github.com/surajkr97/resume.git
   cd resume
   ```

2. Open `resume.tex` in any LaTeX editor (e.g., VS Code).

3. Compile using `pdflatex` or an IDE (e.g., Overleaf).

4. Generate the PDF:
   ```bash
   pdflatex resume.tex
   ```

5. Find the output as `output.pdf`.
